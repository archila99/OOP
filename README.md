# Object-Oriented Python (OOP) Learning

This repository contains my practice and learning exercises for **Object-Oriented Programming (OOP) in Python**.  
It includes examples of **classes, inheritance, multiple inheritance, polymorphism, method overriding, and encapsulation**.

## 🧩 Purpose

The purpose of this repository is to:

- Strengthen my understanding of OOP principles in Python
- Practice writing clean, reusable, and maintainable code
- Experiment with real examples that demonstrate class design and interactions
- Prepare for technical interviews and coding projects


## 🔑 Key Concepts Covered

- **Classes & Objects**: Creating classes, initializing objects, using instance attributes and methods
- **Inheritance**: Single, multiple, and multilevel inheritance
- **Method Overriding**: Extending parent methods with `super()`
- **Polymorphism**: Implementing different behavior using the same interface
- **Encapsulation**: Using private and public attributes and methods
- **Practical Examples**: Mini-projects with Rabbit, Predator, and Creature classes
